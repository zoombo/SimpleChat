Чат который должен запускаться без установки и работать в локальной сети.
Работать может в 4х режимах. 
1) Классический режим, когда один пользователь выставляет состояние "Server" и говорит свой IP 
другому пользователю, который в свою очередь выставляет состояние "Client" и указывает полученный IP.
2) Являет собой тоже что и 1й только добавляется опция "Use ID" в поле которой указывается заранее 
оговоренный ID (любой набор букв и цифр) и таким образом клиент при подключении отправляет серверу 
хеш ключевой фразы, сервер сравнивает хеш своей ключевой фразы с полученным хешем и если они совпадают - происходит соединение.
3) Автопоиск в локальной сети. Пользователь просто выставляет режим "AutoSearch" (и по желанию порт если 
пользователь знает что это такое, если нет используется стандартный 55555) и программа сама осуществляет поиск 
в локальной сети по определенному алгоритму.
  Алгоритм поиска собеседника в локальной сети.
    запуск(автопоиск)
    while not соединение_удалось:
        Отправляем_широковещательный_запрос(порт)
        кто-то_ответил = принимаем_ответы()
        if кто-то_ответил:
            соединение_удалось = соединение_с_ответившим() 
        else:
            сами_становимся_сервером()
            соединение_удалось = слушаем_порт_случайное_количество_секунд(порт, случайное_число_секунд_от_10_до_60)

4) Фактически представляет собой 3й режим + 2й.
